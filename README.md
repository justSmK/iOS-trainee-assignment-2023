## Технические детали

- Язык программирования: [Swift](https://www.swift.org)
- Архитектура: MVP (вариант с MVC можно увидеть, если выбрать ветку [`mvc-architecture`](https://github.com/justSmK/iOS-trainee-assignment-2023/tree/mvc-architecture))
- Интерфейс: [UIKit](https://developer.apple.com/documentation/uikit)
- Список товаров: [UICollectionView](https://developer.apple.com/documentation/uikit/uicollectionview) с [DiffableDataSource](https://developer.apple.com/documentation/uikit/uicollectionviewdiffabledatasource) и [CompositionalLayout](https://developer.apple.com/documentation/uikit/uicollectionviewcompositionallayout) (MVC: [UICollectionView](https://developer.apple.com/documentation/uikit/uicollectionview) с [UICollectionViewFlowLayout](https://developer.apple.com/documentation/uikit/uicollectionviewflowlayout))
- Загрузка данных: [URLSession](https://developer.apple.com/documentation/foundation/urlsession)

## Установка 

1. Выберите версию проекта с нужной архитектурой (по-умолчанию в ветке [`main`](https://github.com/justSmK/iOS-trainee-assignment-2023) архитектура MVP)
2. Клонируйте или скачайте репозиторий (https или ssh):

   ```
   https://github.com/justSmK/iOS-trainee-assignment-2023.git
   ```

   ```
   git@github.com:avito-tech/iOS-trainee-assignment-2023.git
   ```
3. Откройте `Avito Tech Assignment.xcodeproj` в [Xcode](https://apps.apple.com/ru/app/xcode/id497799835?l=en-GB&mt=12)
4. Соберите и запустите проект на симуляторе iOS или реальном устройстве
